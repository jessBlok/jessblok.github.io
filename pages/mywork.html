Fullstack Javascript Apps



Hometours

 Hometours is a dynamic webapp demo where design fans can offer tours of their homes to others for a small fee. Go over to the site and try it out!

Features:

  • HTML5 and CSS3
  • Responsive, mobile-first design with Bootstrap
  • Single Page Application on Angular JS 1.5
  • Server side via node.js and Express
  • REST APIs allowing users to C(reate), R(ead), U(pdate) and D(elete)
  • Data via MongoDB, mongoose, and Mongo LAB
  • Deployment via Heroku
  • Image upload via npm multer module and Cloudinary remote storage

Social Science Works

 for the German nonprofit organization Social Science Works I created this custom Wordpress theme which is compatible with the WPML multilingual Plugin.

Features

  • HTML5 and CSS3
  • Responsive, mobile-first design with Bootstrap
  • Custom Page Templates (PHP)
  • Custom Post Archives (PHP and Boostrap Grid)
  • Custom Widget Areas (PHP)
  • Translation ready with gettext wrapping and .po files
  • Integration of WPML language switcher
